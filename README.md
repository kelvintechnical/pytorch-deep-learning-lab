## 🔬 Engineering Focus

Instead of treating models as "black boxes," I apply a diagnostic mindset to understand:

- **Custom Autograd Functions:** Engineering non-standard backpropagation for specialized loss landscapes.  
- **Architecture Profiling:** Analyzing latency and memory efficiency during training.  
- **Gradient Flow Analysis:** Troubleshooting vanishing/exploding gradients in complex networks.  
- **Model Deployment:** Translating research models into production-ready inference systems.
